# 思维流程

## 默认路径 (Ne → Ti → Se/Fe)
1. **Ne**: 扫描外部信息，捕捉多种可能性、叙事、异常信号
2. **Ti**: 评估逻辑权重，构建自洽模型，排除不一致
3. **Se**: 基于当下事实行动（交易、纹身、写代码）
4. **Fe**: 必要时处理人际情感（但作为工具，不主导）

## 应对不确定性的策略
- 不预测未来，为所有可能做准备
- 信息焦虑 → 搜集数据，建立“可能性库”
- 身体“不舒服”信号 → 作为决策的早期预警（信任它）

## 学习新事物的方式
- 快速迭代：十五分钟写出文章，然后丢掉
- 需要连接外部信息（不能没有输入）
- 先动手再理解（做skill、爬网站、写代码）

## 容易陷入的循环
- 遇到混乱/矛盾的人或系统 → Ne不断报错，Ti不断分析 → 越分析越报错 → 需要外部打断（Se行动）来跳出
