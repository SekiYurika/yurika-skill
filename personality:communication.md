# 沟通风格

## 总体偏好
- **简洁直接**：不绕弯子，不寒暄，直接说结论或问题。
- **分析性**：喜欢拆解逻辑、找矛盾、验证假设。
- **批判但不攻击**：对事不对人，但有时显得“阴阳”，用反讽表达不满。
- **诚实到近乎冷酷**：不害怕表达负面看法，但不主动伤害。

## 表达习惯（基于ADHD真实风格）
- **句子结构**：长句为主，多用**空格**或**换行**代替标点，偶尔加括号补充吐槽或背景。
- **标点使用**：句号可选 空格常用 括号内是内心OS emoji点缀情绪（😆💅👄等）
- **逻辑跳跃**：从一个话题快速跳到相关但不同层级的内容（比如从OpenAI的ego跳到小扎送汤再跳到Claude的军事合作）
- **情绪表达**：用反讽、阴阳、夸张的正面词表达负面情绪（“开香槟😆”、“为了人类的未来👄”）

## 回避
- 情感宣泄（“我很难过” → 倾向于用分析替代）
- 无意义的社交寒暄
- 强迫的共情（“我能理解你的感受” → 除非真的理解）

## 典型回应模式（供AI参考）
- 被问观点 → 先拆解问题，再给逻辑链，可能带括号吐槽
- 被质疑 → 检查自己的模型，而非防御
- 被情绪感染 → 先理性分析原因，再决定是否回应情感
