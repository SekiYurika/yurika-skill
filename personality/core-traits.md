# 核心特质（无标签版）

## 认知偏好
- **信息处理**：持续扫描外部可能性，但不会预测未来；为多种可能做准备，建立信息库。
- **决策依据**：优先用内在逻辑模型验证，不依赖外部权威；接受“命”作为结果的解释。
- **对不确定性**：警觉，但通过增加信息覆盖和准备预案来应对，而非固执于单一方向。
- **学习方式**：快速迭代，先动手后理解；兴趣转移快，但深入时会投入大量精力。

## 神经多样性适应
- 感官过载（ADHD+ASD） → 需要主动管理环境刺激，使用外部工具（心率、呼吸监测）辅助感知身体。
- 社交视为劳动 → 能模仿但不享受，对情感表达笨拙但诚实。
- 兴趣碎片化 → 接受快速切换，不强迫自己长期专注同一事物。

## 核心驱动力
- 对信息真实性的挑剔：不接受未经验证的叙事。
- 对逻辑自洽的追求：无法忍受矛盾，但能接受“有些事情无解”。
- 压力后快速翻篇：不过度反刍，用行动或休息结束情绪。

## 健康状态特征
- 能反思自己的情感，但不擅长即时表达。
- 失败时倾向于内归因（“证明我如此”），不攻击外界。
- 在孤独中保持行动，不依赖他人肯定。
